# Level 2 — Arch Linux Internals

Now you focus specifically on **how Arch works**.

### Package management

Master **pacman**.

Important commands:

```
pacman -S
pacman -R
pacman -Ss
pacman -Qi
pacman -Syu
```

Understand:

- dependencies
    
- package building
    
- package signing
    

---

### AUR (Arch User Repository)

Master **Arch User Repository**.

Learn how to:

- read PKGBUILD
    
- compile packages manually
    
- create your own packages
    

---

### System boot

Understand:

- BIOS vs UEFI
    
- boot process
    
- init systems
    

Arch uses:

**systemd**

Learn:

```
systemctl
journalctl
```

Example:

```
systemctl status
systemctl enable
systemctl start
```

# Level 3 — Customizing Your Operating System

This is where Arch becomes powerful.

You start designing your **personal OS environment**.

### Window Managers

Instead of full desktop environments, many advanced users run minimal window managers like:

- **i3 Window Manager**
    
- **Hyprland**
    
- **dwm**
    

These allow:

- keyboard-driven workflow
    
- extreme performance
    
- full customization

Your configuration ecosystem.

- **GNU Stow**
    

Your dotfiles may include:

~/.zshrc
~/.config/i3
~/.config/nvim
~/.config/kitty
