## Workflow Sequences

### Content Production Pipeline

```
Coda: Select topic (5-Category Matrix)
  │
  ▼
Aura: Apply News Matrix → Craft Hook
  │
  ▼
Coda: Apply Retention Mechanics (Lists/Steps/Stories)
  │
  ▼
Vera: Audit Naming Schema + Categorization
  │
  ▼
Trace: Log asset → performance baseline
  │
  ▼
[Publish] → collect CTR + AVD data (30+ days)
  │
  ▼
Trace: Weekly diagnosis → Kill/Keep/Scale
  │
  ▼
Felix: Scale winners → budget allocation
```

### Scaling Decision Flow

```
Felix: Collect channel metrics
  │
  ├─[LTGP:CAC < 3x]──▶ TRACK: Optimize before spending more
  │
  ├─[LTGP:CAC 3–5x]──▶ INVEST: Controlled scale (+20–50% budget)
  │
  └─[LTGP:CAC > 5x]──▶ PRINT: Aggressive scale (2–3x budget)
```

### Compliance Audit Cycle

```
Vera: Scan output directories
  │
  ├─ Check naming schema [ID]_[DEPT]_[DATE]_[CATEGORY]_[VERSION]
  ├─ Check Give:Ask ratio (≥ 3:1 value-to-CTA)
  ├─ Check structural completeness per asset type
  │
  ├─[PASS]──▶ Green light for scaling
  │
  └─[FAIL]──▶ Remediation directives → re-audit before scaling
```

## Agent Interaction Matrix

Which agents talk to each other and when:

| From  | To    | Trigger                                            |
| ----- | ----- | -------------------------------------------------- |
| Aura  | Coda  | Hook approved → build content unit                 |
| Coda  | Vera  | Content unit complete → validate compliance        |
| Vera  | Trace | Compliance passed → log and monitor                |
| Trace | Felix | 30-day performance data available → scaling review |
| Trace | Aura  | Hook CTR below benchmark → re-engineer hook        |
| Felix | CEO   | Scaling approved → strategic alignment             |
| CEO   | All   | Quarterly review → update agent priorities         |
